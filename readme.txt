== Description ==

Reviews Tap helps businesses collect, monitor and manage reviews across a range of online platforms.

= General Features =

* Automatically request a rating from new subscribers
* Bulk request ratings from existing subscribers
* Display reviews on website through shortcode or Widget

= WooCommerce Features =

* Automatically request a rating from new customers
* Bulk request ratings from existing customers

= Supported Review Sources =
* Google Local Business
* Facebook Recomendations
* Tripadvisor
* Yelp (Coming Soon)
* Capterra (Coming Soon)
* Wongnai (Coming Soon)


== Installation ==


1. Unzip reviewstap.zip and upload the folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add Your Reviewstap API key on the Settings page
